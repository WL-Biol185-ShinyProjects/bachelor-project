# Will You Accept This Rose? — A Bachelor & Bachelorette Data Analysis App

## Project Summary
 
*Will You Accept This Rose?* is an interactive R Shiny web application that explores the data and statistics behind the long running reality shows The Bachelor and The Bachelorette. We have used contestant data to show hometowns, predict casting likelihood, analyze contestant demographics, explore cultural patterns, and create fun interactive games to predict the users likelihood of success on each show.
